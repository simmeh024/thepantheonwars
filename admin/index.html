      Dispatch Translations

      Rule-based drafts turn raw commits into clear end-user summaries. Review or edit each draft before approving it for the public Development Dispatches page.

      Commit title Status Confidence score Confidence is explainable evidence, not a writing-quality grade: recognised subject (25), reviewed dictionary (10), commit intent (30), body context (10), safe changed-file scope (20), and optional local semantic support (5, from either spaCy's domain match or a strong sentence-embedding match against approved past Dispatches). High confidence needs at least 65% and two independent signals; two independent formatter rules can establish the 65% floor. A RapidFuzz concept match always requires editorial review. A strong embedding match never changes the draft text itself -- it only appears as a "Similar past Dispatch" reference for you to compare against.
